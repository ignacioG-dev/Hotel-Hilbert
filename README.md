
# Hilbert Hotel

## Descripción
Este es el repositorio del proyecto *Hilbert Hotel*, un programa de gestión de hotel escrito en C. El proyecto se centra en administrar las reservas de habitaciones, mantener un registro de los huéspedes y proporcionar información sobre estadísticas de ocupación.

## Requisitos
- Compilador de C
- Biblioteca estándar de C
- Sistema operativo compatible con C (Windows, Linux, etc.)

## Características Principales
- Registro de huéspedes con información detallada, incluyendo nombre, email, género y plan.
- Administración de habitaciones por piso y número de habitación.
- Generación de informes de ocupación y estadísticas de huéspedes.
- Gestión de planes de alojamiento básico y premium.

## Menú de Opciones
El programa presenta las siguientes opciones en su menú principal:

1. **Ingresar Datos:** Permite ingresar información de nuevos huéspedes.
2. **Huespedes:** Proporciona opciones para gestionar la lista de huéspedes, como ver la lista y desocupar habitaciones.
3. **Historial de Huéspedes:** Genera un informe de la ocupación y estadísticas por piso.
4. **Información de Planes:** Muestra información sobre los planes de alojamiento disponibles.

## Datos de Acceso
El programa tiene un sistema de inicio de sesión simple que requiere un nombre de usuario y una contraseña. Puedes personalizar estos datos en el código fuente.
